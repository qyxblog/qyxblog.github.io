怎么免费搭建博客网站呢？

📖前言

没有错，没有错！MaxLen来教大家搭建博客站了！不需要任何编程基础，人人都可以有一个个人网站！

本教程主要讲的是Hexo，其它工具也类似，部署平台是Github。教程适用于Windows端，利用手机建站请另寻高明（也不会有人在手机上码字吧）。

🔧准备

我们需要三个东西：

  1. Node
  2. Git
  3. Github账号

如果没有Github账号请注册一个~

只不过啊，需要注意：以后你网站的网址就是你的Github账号名+github.io

所以注册的时候建议简短一些，并且好记一些，比如说我的Github账号就叫“qyxyyds”，哎~是不是很好记。

如果你的网络环境不能访问Github，请自行寻找访问的 办法 魔法（WinXray是个很好的电脑工具）
