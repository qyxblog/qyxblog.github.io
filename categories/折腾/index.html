Win端完美深色2，带你打出趋于完美的组合拳！

🖊前言

我之前写过一篇让Windows自动切换深色和浅色模式的文章，在那篇文章里，我们使用的是计划任务，但是它有一个缺点：比如说我把自动切换深色的时间设在18:00，那么如果我在18:00没开电脑，而在18:00后开了电脑，那么就不会自动切换深色模式，浅色模式的切换，缺点也是如此。

那么有没有更好的方案？

🤜第一式：Windows-Auto-Night-Mode

Windows-Auto-Night-Mode是一个在Github上开源的软件，意味着纯净、免费、不作恶。

安装界面多语言可选，但是没有中文，不过不用担心，安装完毕后，主界面是中文的。

你可以通过自定义时间或者日出至日落来自动切换深色和浅色。

点击应用程序，可以进一步自定义

最棒的是，它支持切换壁纸！我可以在白天使用浅色壁纸，在夜晚自动切换为深色壁纸！

⚔第二式：Twinkle Tray

如果你是笔记本用户，你可以按动笔记本上的亮度切换，以此调整亮度

但如果你是台式机用户…是的，我知道你不想打开显示屏复杂的菜单去调亮度，所以就有了Twinkle Tray

和Windows-Auto-Night-Mode一样，它也在Github上开源。

首次使用，它会躲在你的托盘里。当你把鼠标移到它的图标上，再滚动滑轮时，电脑屏幕的亮度也会跟着改变。

点击托盘图标也可以调亮度。

它还支持根据时间自动切换显示屏的亮度

当然也少不了快捷键控制亮度，笔记本般的体验！

值得一提的是，Twinkle Tray对多显示器有着较好的支持。

📕总结

有了这两款应用，你的电脑基本就可以实现完美的Dark Mode了！

如何让Windows 10自动切换深色模式

🚗引入

Windows 10中，你可以手动在设置中切换浅色or深色模式，但不能自动切换

Windows 10中，还有一个夜间模式，但它并不是人们所理解的“夜间模式”，我更愿意称他为“护眼模式”。

那有没有办法让Windows 10自动切换深色模式呢？有两种方法。

⏲这个被你忽视的小功能：计划任务

二分之一

打开任务计划程序，如果你不知道它在哪里可以找到，你可以使用Windows+S打开搜索找到它但这里我们依然提供它的路径：控制面板 > 所有控制面板选项 > 管理工具

点击”创建基本任务”

在启动程序中，输入reg.exe，而参数则输入：

add HKCU\SOFTWARE\Microsoft\Windows\CurrentVersion\Themes\Personalize /v SystemUsesLightTheme /t REG_DWORD /d 0 /f

唉？你以为好了？其实还没好哦！这只是切换了Windows为深色，而应用还不是深色哦，依法炮制，我们再创建一个计划程序，在启动程序中同样输入 reg.exe，参数输入：

add HKCU\SOFTWARE\Microsoft\Windows\CurrentVersion\Themes\Personalize /v AppsUseLightTheme /t REG_DWORD /d 0 /f

OK，至此，自动深色部分已经完成，接下来我们还需要让它自动变为浅色

其余的二分之一

我们依然依法炮制：在启动程序中同样输入 reg.exe，参数输入：

add HKCU\SOFTWARE\Microsoft\Windows\CurrentVersion\Themes\Personalize /v SystemUsesLightTheme /t REG_DWORD /d 1 /f

再创建另一个：在启动程序中同样输入 reg.exe，参数输入：

add HKCU\SOFTWARE\Microsoft\Windows\CurrentVersion\Themes\Personalize /v AppsUseLightTheme /t REG_DWORD /d 1 /f

二分之二

至此，自动开启or关闭深色模式的第一种方法就结束了！

更加快速简单的方法：Windows-Auto-Night-Mode

这玩意是开源的，也就是说它可以保证绝不耍流氓，代码透明、人人可见，绝无病毒！

值得一提的是：MaxLenの博客站和随机波动博客站也是在Github上开源的哦！

这个应用在此就不多赘述，详见请看少数派的测评报告

下载地址：Github （国内直链稍微困难）

图片来源：少数派

👍总结

OK，这篇教程到这里就结束了，但它们并不完美无缺，想要无割裂且完美的体验，只能等待微软官方的跟进。这个功能能否在Windows 11正式版上实现？以微软的尿性，我觉得很难。
